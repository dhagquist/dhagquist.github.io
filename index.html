Web Maps

Golf Courses in Ohio (2017)

This project utilizes Python, the Google Places API, QGIS, the QGIS2Web plugin, and Leaflet. HTML, CSS, and Leaflet JS were customized manually. Hover over the green dots to view the golf course addresses.

Backstory:
I needed to produce a list of golf courses in Ohio with their addresses. All I could find online, however, were data sets with only the name of the course, the city, and the state (not the full address). So, I geocoded the data set in QGIS using the MMQGIS plugin which generated latitude and longitude coordinates. Using these coordinates I was able to write a script in Python and using the Google Places API in order to get the full address information.
